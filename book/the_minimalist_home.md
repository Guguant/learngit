[美] Joshua Becker / Eric Stanford / 著

苏西 / 译

======

极简主义的定义：

有意识地提升我们最珍视的事务的地位，把其余一切令我们分心的事务清理掉。

======

停泊在港湾里的船是安全的，但那并非它的使命。

======

======

信仰、家庭、还有朋友，这些生命中最重要的事始终没有变。

======

我没有只是坐在那儿期待或做梦，我采取了行动。

======

极简主义是一条道路，通往这样一个目的地：用新近发现的热情、使命和富余出来的资源，去追求生命中最重要的事。这样的人生是不是很棒？

======

心无挂碍，说走就走。

======
